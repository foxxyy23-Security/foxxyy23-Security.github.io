SMB Enumeration (Port 445)

Listing Shares

smbclient

smbclient -L $target -N  # Null Session
smbclient \\$target\$share -U <username>

SMBMap

smbmap -H $target

CrackMapExec / NetExec

netexec smb $target --shares -u '' -p ''

User Enumeration

Using NetExec (formerly CrackMapExec) to find users and brute force RIDs.

netexec smb $target -u '' -p '' --users
netexec smb $target -u '' -p '' --rid-brute

Connecting to a Share

smbclient //$target/<share_name> -U <user>

Impacket

Dump domain info via SAMR:

samrdump.py $target
